# Garden Blocks Playable

Cozy garden-themed re-skin of Color Block Jam, built as a self-contained
playable ad. Drag flower planters across a 6×8 wooden board and exit them
through their matching-colored garden gate.

Built from the Playables Blitz monorepo — this repo only hosts the compiled
single-file HTML for public preview via GitHub Pages.

**Live demo:** see GitHub Pages tab in repo Settings.

The whole game (Three.js scene, atlased 3D flower meshes, audio, MRAID
compliance) is base64-inlined into `index.html` (~3.2 MB). No external
network requests at runtime.
